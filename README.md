## markdown book
markdown book是一个管理、编辑markdown文件，并可以将之生成易于阅读的在线书籍的工具。

## 功能
- [x]  粗体
- [x]  斜体
- [ ]  标题（1- [ ] 6级标题）
- [x]  下划线
- [x]  中划线
- [x]  标记
- [x]  上标
- [x]  下标
- [x]  居左
- [x]  居右
- [x]  居中
- [ ]  引用
- [ ]  有序列表
- [ ]  无序列表
- [ ]  链接
- [ ]  图片
- [ ]  代码块
- [x]  表格
- [ ]  上一步
- [ ]  下一步
- [ ]  删除
- [ ]  预览
- [ ]  全屏
- [ ]  沉浸式阅读

## 插件
- markdown-it.js
- highlight.js
